= calc_sun

Home    :: https://rubygems.org/gems/calc_sun
Code    :: https://github.com/DouglasAllen/calc_sun
RDoc    :: http://www.rubydoc.info/gems/calc_sun/0.1.1
Issues  :: https://github.com/DouglasAllen/calc_sun/issues


== DESCRIPTION:

Calculate Sunrise and Sunset. Now uses native C extension.
An example is supplied to show gem usage.

== INSTALL:

  $> gem install calc_sun

== DOWNLOAD:

  $> git clone https://github.com/DouglasAllen/calc_sun.git
  $> cd calc_sun
  $> bundle install
  $> bundle exec rake install


=== Build with DEVKIT

   cd to devkit dir and run devkitvars.bat

   or make a batchfile to start your envirnment with 'run devkitvars'
   cd to calc_sun dir
   Rake -T

== DEVELOPERS:

After checking out the source, run:

  $ bundle exec rake newb

This task will install any missing dependencies, run the tests/specs,
and generate the RDoc.

  n/a

== SYNOPSIS:

=== start irb

  $ irb --simple-prompt

=== enter Code

        require 'calc_sun'
        cs = CalcSun.new
        require 'date'

        lat = 51.4770228
        lon = -0.0001147
        day = Date.parse('2016-12-25')
        day.jd
        jd = cs.df2000(day, lon)
        jd
        cs.t_rise(jd, lat, lon)
        cs.t_mid_day(jd, lat, lon)
        cs.t_set(jd, lon, lat)
        DateTime.jd(day.jd + cs.t_rise(jd, lat, lon) / 24.0)
        DateTime.jd(day.jd + cs.t_mid_day(jd, lat, lon) / 24.0)
        DateTime.jd(day.jd + cs.t_set(jd, lat, lon) / 24.0)
        Date.jd(day)
        # new FEATURES version 0.1.2
        jd = day - CalcSun::DJ00 - lon / 360.0
        cs.rise(jd, lat, lon)
        cs.noon(jd, lat, lon)
        cs.set(jd, lat, lon)
        day_array = ['2016-4-14', '2016-6-15', '2016-8-31', '2016-12-25']
        (0..3).each do |i|
          day = Date.parse(day_array[i])
          jd = cs.df2000(day, lon)
          p day.jd
          cs.rise(jd, lat, lon)
          cs.noon(jd, lat, lon)
          cs.set(jd, lat, lon)
        end
